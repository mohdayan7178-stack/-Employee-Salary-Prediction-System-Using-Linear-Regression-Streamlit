# Employee Salary Prediction System Using Linear Regression

## Overview
This project predicts an employee's salary based on years of experience using a Linear Regression Machine Learning model. The application is built using Python and Streamlit.

## Features
- Predict salary based on years of experience
- Interactive Streamlit web interface
- Linear Regression model
- Fast and simple UI

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

## Project Structure

```
app.py
salary_model.pkl
requirements.txt
README.md
```

## Installation

```bash
git clone https://github.com/yourusername/Employee-Salary-Prediction-System-Using-Linear-Regression-Streamlit.git

cd Employee-Salary-Prediction-System-Using-Linear-Regression-Streamlit

pip install -r requirements.txt

streamlit run app.py
```
